# Firmware Artifacts

Validated GX12 firmware:

- `R2X-7D8.BIN` - composite CDC/HID, 2x trainer-resolution GX12 firmware.

Experimental tester firmware:

- `experimental-tx16s\` - untested TX16S/TX16S MK3 candidates. These are not
  GX12 firmware and are not guaranteed.

The matching GX12 firmware source is staged in the public repo under
`firmware\edgetx-gx12-2.11.0`. Rebuild notes and hashes are in
`docs\firmware-source.md`.
